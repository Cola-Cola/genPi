# Вычисление числа ПИ методом Монте-Карло

---
Реализация подготовлена на трех языках: C, Python, Go

Для С: 
`g++ GenPi.c -o GenPiC`
`.\GenPiC.exe`

Для Go:
`go run GenPi.go`

Для Python:
Python 3.7 recommended
`pip install numpy`

Если несколько версий Python - принудительный запуск pip нужной версии:
`pip37 install numpy`
`python GenPi.py`
